# SpellForce Pak Tool

A modern, cross-platform utility for unpacking and packing `.pak` archives for the game **SpellForce 1/2** (and its expansions).

## Features
* **Cross-platform**: Works on Windows, macOS, and Linux.
* **Unpack**: Extract individual `.pak` files or batch extract an entire directory.
* **Pack**: Create valid SpellForce `.pak` archives from a folder.

## How to Build
1. Install the [.NET 10 SDK](https://dotnet.microsoft.com/download).
2. Clone the repository.
3. Run the following command in the `SFPakTool/SFPakTool` directory:
   ```bash
   dotnet run
   ```
